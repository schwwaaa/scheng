# Max patches (OSC controllers)

These patches send OSC messages to `127.0.0.1:9000` using the SDK convention:

- address: `/param/<name>`
- arg: one float

Patches included:
- `feedback_orb.maxpat`
- `feedback_pingpong.maxpat`
- `temporal_slitscan.maxpat`

Also included (as reference):
- `scheng-OSC-buffer.original.maxpat`
