standalone binary workflow:
1. Export graph JSON → my_effect.json
2. cargo run -p scheng-export -- my_effect.json --out my_effect/
3. cd my_effect && cargo build --release
4. ./target/release/my_effect --osc-port 57120 --fullscreen

OSC/MIDI works identically — same addresses as the bridge.
Cross-compiles to Linux aarch64 (Raspberry Pi 5, Jetson Orin).
